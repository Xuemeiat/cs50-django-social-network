# cs50-django-social-network
Social network web application built with Django and JavaScript as part of CS50 Web Programming.

## Features
- User registration and login
- Create and edit posts
- Like posts
- Follow/unfollow users
- View following feed
- Pagination
- AJAX-based interactions

## Technologies
- Python
- Django
- JavaScript
- HTML & CSS
